# 知识点溯源与低样本能力评分设计

> **D01同步**：本文的学习者评分部分改为 Pandas 五维正式上层画像。AI直接评分可观察维度并引用 Evidence；无证据为 `unverified`，失败保留旧版本。确定性 `mastery/confidence/KnowledgeState`仍是路径唯一事实。CIDPP五维仅评价教学内容，严格分离。

## 1. 目的与状态

知识点级溯源是 `[MVP]` 直接纳入能力，用来限制生成来源、审计 claim 并计算核心覆盖；本文的学习者能力五维评分已由 D01 冻结为 Pandas 上层画像，AI直接评分可观察维度，不能改写确定性知识点底座。高级雷达图可降级隐藏，但画像数据不能删除。02、03号中的内容质量 CIDPP 只用于讲义、题面和反馈优化，不属于学习者能力评分，也不进入 `mastery` 公式。

## 2. 溯源模型

```ts
interface SourceAnchor {
  sourceAnchorId: string; sourceId: string; uriOrFile: string;
  section?: string; lineStart?: number; lineEnd?: number;
  excerptHash: string; version: string; license?: string;
}
interface KnowledgePointRecord {
  knowledgePointId: string; title: string; prerequisites: string[];
  sourceAnchorIds: string[]; core: boolean; activityIds: string[];
}
interface ClaimRef { claimId: string; sourceAnchorIds: string[]; status: "supported"|"unverifiable"|"rejected"; }
```

`source_map` 是已有容器；MVP 为每个 Pandas 核心点补稳定 ID、官方文档片段定位、版本和摘录哈希。生成请求只允许当前 Profile revision 的 anchor 白名单；输出每个原子 claim 的 `claimRef`。Hunter 逐 claim 核查，Judge 只能接受 supported 或显式人工复核的 unverifiable，不能由模型自行扩大来源。

## 3. 数据流与失败

构建时校验文件存在、哈希、许可和知识点引用；生成时注入来源摘录而非整库；审核时记录 claim 状态；发布时阻止缺 anchor 的核心事实。来源删除、版本哈希变化或引用不存在会使包进入 `draft_invalid`，旧 `active` 保留可用。无法确认的 claim 进入人工队列，不悄悄标为正确。

## 4. 低样本能力状态

每个能力维度记录 `mastery`（0–1）、`confidence`（0–1）、`evidenceCount`、`lastEvidenceAt`、`sourceActivityIds`。初始值来自诊断题的确定性映射；活动证据采用带衰减的加权更新：独立实操 > 代码补全 > 客观题 > 自陈/提问软证据。一次错误只能降低对应点的 mastery，不级联改写已完成节点。

```text
new_mastery = clamp(old + weight * direction * confidenceFactor, 0, 1)
new_confidence = min(1, old + evidenceReliability / (evidenceCount + 1))
```

权重、阈值和映射表属于 Profile 确定性资产并版本化；证据不足时输出 `insufficient_evidence`，PathEngine 选择短诊断或低风险活动，不伪造精确分数。

## 5. CIDPP 参考边界

学习者能力评分可离线比较理解、应用、分析、创造、反思五维与 Skill-Tree 的映射；第3周根据重复评价一致性、证据忠实度、决策价值、延迟和路径合法性决定是否启用方案B，任一门禁失败则只报告方案A的知识点 `mastery/confidence`。雷达图仅在方案B通过后作为条件展示，不得反过来驱动路径。内容质量 CIDPP 的五维评价由02、03号规定，服务于资料包构建，不作为学习者分数。

## 6. 测试与验收

固定诊断、全对/全错、矛盾证据、重复提交、低样本和评测器异常均有单元测试。验收要求：所有分数可由 evidence 重算；软证据不得直接标记掌握；核心覆盖率分母为冻结的 `core=true` 知识点；路径合法率 100%。
